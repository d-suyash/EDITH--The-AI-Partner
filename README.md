# EDITH - THE AI PARTNER

The EDITH is an intelligent agent that has been developed with the objective of using the power of Natural Language Processing and Supervised Machine Learning.

# **Key AI Features**
**Intelligent Agent Structure**: It has been designed to work as a "Goal-Based Agent" which can understand the input of the user and act accordingly.

**Understanding user's request**: It uses the Probability-based Categorizer to understand the request of the user.

**Feature Extraction**: It uses the Count Vectorizer tool to extract data from the text.

**Knowledge Retrieval**: It uses external sources such as Wikipedia and Google to answer user questions such as "Who is" and "What is."

# **How It Works**
**Percept**: The user enters the command in natural language.

**Transformation**: It is converted into a mathematical vector.

**Inference**: It uses the Probability-based Categorizer to determine the need of the user.

**Action**: It performs the action based on the Python script.

# **Tech Stack**
**Programming Language**: Python 3

**Machine Learning**: Scikit-learn

**Vectorization**: Count Vectorizer

**System Libraries**: os, sys, webbrowser, datetime

# Installation and Usage

Install the ML brain:

Run **pip install scikit-learn** in the terminal of your compiler to download scikit-learn.

Run the Agent:

Run the code in the compiler.

Interact: Type your command at the USER >> prompt and press Enter.- THE AI PARTNER
